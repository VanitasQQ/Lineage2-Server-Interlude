Глубинный Празднующий Андреас:
Скоро мы проведем тёмную мессу в честь Грен Кейна и Шилен, матери пропасти. Эта месса - самая священная религиозная церемония нашей расы. Человеческие волшебники из Башни Слоновой кости, которые посетили прошлогоднюю массу, были поражены размерами и великолепием мессой. Месса этого года будет не менее великой. Духовенство храма очень занято приготовлениями. Есть несколько изделий, необходимых для мессы, которые должна быть доставлены из деревни. Если мы не справимся, мне возможно потребуется ваша помощь, чтобы доставить те изделия для меня.

(Задание для Темных Эльфов уровня 2 или выше.)
